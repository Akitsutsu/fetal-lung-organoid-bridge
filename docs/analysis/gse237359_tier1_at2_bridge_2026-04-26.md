## Summary

`GSE237359` is the clearest current bridge-side **Tier 1** dataset for
AT2-related distal alignment from an independent fetal-primary organoid system.

Its main bridge value is not native tissue context or mature AT1 closure.
Instead, it does three narrower jobs well:

1. it shows that a fetal-primary HDBR-derived AT2 organoid can retain a strong
   `SFTPC` / `HOPX` / `AQP5` program;
2. it independently reinforces the D-0008 line that low `AQP5` is primarily an
   **iPSC/hESC-derived organoid-system boundary**, not a universal organoid
   boundary;
3. it strengthens Tier 1 without collapsing epithelial success into positive
   mesenchymal niche support.

## Why This Dataset Matters For Tier 1

The Tier 1 question is not "does this close mature AT1 identity?" It is
"can an organoid system align to a fetal distal epithelial / AT2-related
program in a biologically interpretable way?"

`GSE237359` answers that question unusually well for three reasons:

- it is fetal-primary rather than iPSC/hESC-derived;
- it comes from the HDBR source bank rather than the UW LDB bank that underlies
  the registered Quach-Farrell spatial anchor and `E-MTAB-8221` Day_0;
- it carries strong alveolar epithelial markers while remaining clearly
  post-budtip.

## Independent Support For The AQP5 System Boundary

The strongest bridge implication of `GSE237359` is the system-boundary point.

- `GSE237359` is HDBR-derived and therefore source-bank-independent from the UW
  LDB-linked fetal-primary references already used elsewhere in the bridge.
- In this HDBR system:
  - `AQP5` = `0.304`
  - `HOPX` = `0.897`
  - `SFTPC` = `1.000`
- In the current bridge interpretation, many iPSC/hESC-derived organoid systems
  remain `AQP5`-low even when full-transcriptome data are available.

The biological consequence is important:

- the bridge should not say that organoids in general cannot carry
  interpretable `AQP5`;
- it should say that **the current low-`AQP5` boundary is concentrated in
  iPSC/hESC-derived systems**.

This is exactly the narrowed D-0008 reading that the bridge wants.

## Relation To Tier 2 And Tier 3

`GSE237359` supports Tier 1 more than Tier 2 or Tier 3.

### What it supports

- strong AT2-related distal epithelial alignment;
- fetal-primary corroboration of `HOPX` / `AQP5` retention;
- a biologically credible organoid-side bridge into
  `alveolar_epithelial_maturation`.

### What it does not support

- native fetal tissue context;
- donor-resolved trajectory;
- full AT1 maturity closure;
- mesenchymal E1-E3 positive-support adjudication;
- independent spatial replication.

The low `AGER` / `PDPN` fractions are useful here because they keep the read
honest. This is not a mature AT1 system masquerading as Tier 1. It is a strong
AT2-related bridge with partial AT1-associated markers.

## Source-Bank And Donor Structure

The donor metadata support independence but also impose limits.

- Souporcell-resolved HDBR donors:
  - `HDBR16392` 34.8%
  - `HDBR15934` 22.6%
  - `HDBR16402` 21.3%
  - `HDBR16011` 2.7%
- doublets: 18.3%
- unassigned: 0.4%

This means:

- the dataset is not a single-donor anecdote;
- it does provide source-bank diversity relative to the UW LDB references;
- but it is still a pooled library with unbalanced donor contribution and is
  therefore not a donor-balanced replication object.

## Bridge Use

The bridge should use `GSE237359` in exactly one main way:

- as a **Tier 1 fetal-primary AT2 bridge** that makes the iPSC/hESC-specific
  `AQP5` boundary more biologically credible and more source-bank-independent.

It should not be used as:

- a registered claim-support row without review,
- a native fetal tissue anchor,
- a budtip reference,
- or positive mesenchymal niche evidence.

## Paper Implication

This dataset helps the paper say something sharper than "organoids are
imperfect."

A more honest line is:

- fetal-primary organoid systems can retain an interpretable
  `SFTPC` / `HOPX` / `AQP5` epithelial program;
- the bridge-side AT1 gap is therefore not just a universal organoid failure;
- it is partly a **system-specific limitation of current iPSC/hESC-derived
  organoid implementations**, while mesenchymal niche closure remains a
  separate unresolved problem.
